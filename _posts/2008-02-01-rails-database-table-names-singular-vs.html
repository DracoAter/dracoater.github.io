--- layout: post title: 'Rails database table names: singular Vs. plural' date: '2008-02-01T00:06:00.008+02:00' author: Juri Timo≈°in tags: - Rails - Ruby modified_time: '2010-08-02T13:28:23.642+03:00' blogger_id: tag:blogger.com,1999:blog-360329120074358364.post-105150718900181032 blogger_orig_url: http://dracoater.blogspot.com/2008/01/rails-database-table-names-singular-vs.html --- Rails < 2.0 : You need to add into config/environment.rb file line 
# Include your application configuration below
ActiveRecord::Base.pluralize_table_names = false #added line
Rails >= 2.0 : You need to add into config/environment.rb file inside block 
Rails::Initializer.run do |config|
  config.active_record.pluralize_table_names = false #added line
end
